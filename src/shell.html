Beyond SU(4) — measure instead of assume

Everything above is a theorem, and only for $SU(4)$ on T²/ℤ₂. For any other spectrum there is no theorem, so the pairings have to be measured. That is what this panel does, and it is the part of this page that a formula cannot replace.

One row per tower: m, q, multiplicity. Both combs are measured directly, so this stays valid for any group and any orbifold.

One row per diagonal slot of the group element your orbifold hands you: twist sign (±1), Wilson-line charge. A centre-parity rule exists only if a unit shift is Weyl-equivalent to a central element.

The rule. Advancing the Wilson line by one period is, up to a Weyl reflection, multiplication by the central element −1 of Z(SU(4)). A representation answers with the scalar (−1)a+2b+3c, so the harmonics carrying the opposite sign are identically absent — not suppressed, absent.
